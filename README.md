# Quizzes

[![CircleCI](https://circleci.com/gh/rage/quizzes/tree/master.svg?style=svg)](https://circleci.com/gh/rage/quizzes/tree/master)

To start the dashboard:

```bash
yarn install
yarn start-dashboard
```

To start the server:
```bash
yarn serve
```

Watch server and common code changes:
```bash
cd packages/backend
yarn run watch-ts
```
